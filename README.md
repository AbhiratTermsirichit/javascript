# สรุปจากหนังสือ  "พัฒนาเว็บแอปพลิเคชั่นด้วย JavaScript"

![cover](images/cover_big.jpg)

## เนื้อหาเกี่ยวกับอะไร
ทั้งหมดจะสรุปเนื้อหาเกี่ยวกับ JavaScript ยุคใหม่ (ES6) พร้อมทั้งซอร์สโค้ดจากหนังสือตามรูปข้างต้นครับ รวมทั้งยังเพิ่มเนื้อหาอื่นที่ไม่อยู่ในหนังสือ เช่น ES7 และอื่นๆ อีกมากมาย  ซึ่งตอนนี้ผมยังเขียนสรุปยังไม่เสร็จดี (เสร็จไปแค่ 1 %) ว่างๆ ก็จะมาอัพเดตใหม่เรื่อยๆ
